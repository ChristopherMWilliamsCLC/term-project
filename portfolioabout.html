ABOUT

My name is Christopher Williams I’m currently a student at the College of Lake County pursuing an associate’s degree in applied science with the web programming option. I currently hold a career certificate for computer programming from the College of Lake County. I’ve decided to stay in school for an associate because I enjoyed the classes I took, and because my programming certificate already completed about 30% of the associates anyways.

I originally started writing computer programs on an old game called Neverwinter Nights which is a D&D based BioWare game that allows players to build and host their own game using Aurora toolset. After being somewhat dissatisfied with playing games other people made, I decided to start making my own game. I joined a Neverwinter Nights discord server and they assisted me with coding concepts until I was able to write my first program.

I figured that since I knew how to write code anyways that I would go to school and try to turn it into a career. I recently landed my first entry level tech job at a place called Code Ninjas where I assist students in code activities using Microsoft MakeCode, JavaScript, and Unity C#. I’m not an expert and I’m still learning things myself. I get a great deal of satisfaction when I’m able to help a student figure out how to do something.

I currently work at code ninjas and also a small local store in Antioch called Old Mill Creek Country Store where I run the cashier, use deli equipment to slice meats and cheeses, clean the store, and take out garbage and card-board boxes. Old Mill Creek Country Store was rated as the best customer service in Antioch with genuine reviews from customers. I’m at the front-end of the store and I can say with certainty that’s all true.

I’m well versed in different coding languages, but I wouldn’t call myself a master in any of them. My personal preference is the C based languages because declaring your data type prevents a ton of debugging right off the bat. I can write code in most of the main languages used in professional environments regardless of my preference. Some of the languages I’ve used are C++, C#, Java, Javascript, Unity C#, Microsoft MakeCode, SQl, Python, HTML, CSS, and NWN Script which is BioWare’s Aurora toolset language based on C.
